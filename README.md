# スキル管理システム

PHP/MySQLを使用したCRUD操作が可能なWebアプリケーション

## 機能

- スキルの一覧表示
- スキルの追加
- スキルの編集
- スキルの削除

## 使用技術

- Linux (Ubuntu on WSL2)
- Apache
- MySQL
- PHP
- HTML/CSS

## 環境構築
```bash
# Apacheインストール
sudo apt install apache2

# MySQLインストール
sudo apt install mysql-server

# PHPインストール
sudo apt install php php-mysql libapache2-mod-php
```

## 学習目的

インフラエンジニア・バックエンドエンジニアを目指して、実践的なスキルを習得するために作成しました。
